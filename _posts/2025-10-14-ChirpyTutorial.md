### 줄 번호가 있는 코드

```javascript
// 줄 번호 자동 표시
function fibonacci(n) {
  if (n <= 1) return n;
  return fibonacci(n - 1) + fibonacci(n - 2);
}

console.log(fibonacci(10));
```

### 시퀀스 다이어그램

```mermaid
---
config:
  theme: redux-dark
```
